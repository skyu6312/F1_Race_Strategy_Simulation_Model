# 🏎️ F1 Race Strategy Optimization Engine (Spa-Francorchamps)

> **XGBoost 기반 다변수 타이어 디그레데이션 모델링 및 격자 기반 룩업 테이블(LUT)을 활용한 초고속 몬테카를로 레이스 시뮬레이터**

본 프로젝트는 F1(Formula 1) 벨기에 그랑프리가 개최되는 가혹한 고속 서킷 **스파-프랑코샹(Spa-Francorchamps, 44Laps)**을 배경으로, 조지 러셀(Mercedes) 차량의 최적 피트스톱 타이밍과 타이어 컴파운드 포트폴리오를 도출하는 엔지니어링 시뮬레이션 시스템입니다. 

실제 2024-2025년 벨기에 GP의 전략적 흐름(정석 1스톱의 타당성)을 수학적·통계적으로 완벽하게 검증해 냈습니다.

---

## 🏗️ 시스템 아키텍처 (System Architecture)

본 엔진은 대규모 시뮬레이션의 연산 오버헤드를 극복하기 위해 **사전 연산 격자 매트릭스(Look-up Table)** 구조를 채택하여, 10,000회의 평행우주 시뮬레이션을 단 1.12초 만에 완수하는 초고속 아키텍처로 설계되었습니다.

[Layer 1: Data & ML Model]
└── 4개년 스파 GP 데이터 정제 (Dry 전용, 마이너 드라이버 컷오프)
└── XGBoost를 활용한 상대적 페이스 지연(Pace Delta) 학습 (R²: 87.20%)
↓
[Layer 2: Pre-computation (Look-up Table)]
└── 5,808가지 모든 가상 주행 환경 격자(Grid) 생성 후 일괄 Predict
└── O(1) 탐색 성능의 다차원 딕셔너리 매트릭스로 메모리 적재
↓
[Layer 3: Simulation Engine (Monte Carlo)]
└── 62.5% 확률 기반의 세이프티 카(SC) 정규분포 난수 이벤트 주입
└── 루프 내 모델 호출 없이 LUT 다이렉트 매핑을 통한 총 레이스 시간 연산

## 🛠️ 핵심 엔지니어링 파이프라인

### Step 1. 매크로 통계 및 데이터 정제
* **역사적 변수 추출:** 스파 서킷의 평균 피트레인 손실 시간(18.88초), 세이프티 카 발생 확률(62.5%), 발생 평균 랩(3.0랩)을 데이터셋에서 추출하여 환경 상수로 정의.
* **데이터 무결성 확보:** 수중용 타이어(`INTERMEDIATE`, `WET`) 데이터를 배제하여 드라이 체제(Regime)로 한정. 누적 랩 수 100바퀴 미만의 마이너 드라이버를 자동 탈락(Cut-off)시켜 범주형 변수의 희소 데이터 과적합(Overfitting) 노이즈를 완벽 차단.

### Step 2. XGBoost 기반 상대적 마모 모델링 (Pace Delta)
* **상대 페이스 변환:** 팀 간의 절대적인 차량 성능 편차 노이즈를 제거하기 위해, 종속변수($y$)를 절대 랩타임이 아닌 드라이버 고유 베이스 페이스 대비 지연 시간인 `PaceDelta`로 변환(Zero-centering).
* **성능 지표:**
  * **설명력 ($R^2$ Score):** **87.20%**
  * **예측 오차 범위 (RMSE):** **±0.539초**
* **인사이트:** 변수 중요도(Feature Importance) 분석 결과 `Stint`, `LapNumber`, `TyreLife`가 최상위권을 차지하며 스파 서킷의 가혹한 디그레데이션 특성을 완벽히 포착. 타이어 관리 성향이 뚜렷한 드라이버(`Driver_HAM`) 및 차종별 열화 패턴(`Team_Haas`) 지표를 유의미한 변수로 유지.

### Step 3. 룩업 테이블 기반 초고속 몬테카를로 시뮬레이터
* **연산 최적화:** 레이스 루프 내에서 무거운 `.predict()` 모델 호출을 완전히 제거. 사전 구축된 룩업 테이블 딕셔너리를 활용해 연산 속도를 100배 이상 단축 (**10,000회 시뮬레이션 소요시간: 1.12초**).
* **확률적 변수 통제:** 세이프티 카 발생 시 피트레인 손실 시간이 12.50초로 줄어드는 Cheap Pitstop 메커니즘과 실시간 노면 온도 변동 리스크 반영.

| Plan C (2스톱 공격): 12랩, 26랩 [M ➡️ M ➡️ S] | 79분 39초 (+12초) | 4735.64초 | 4813.24초 |

### 💡 핵심 전략 브리핑
1. **Plan A의 타당성:** 스파 서킷은 가혹한 High-Deg 특성을 가졌으나, 피트레인을 한 번 더 통과하며 버리는 18.88초의 실전 패널티(Pit Loss)가 너무 강력하기 때문에 새 타이어의 그립 이득으로 이를 역전할 수 없음. 따라서 **1스톱 전략이 수학적 최적해**임을 증명.
2. **역사적 검증:** 2024년 조지 러셀의 1스톱 우승 도박(무게 미달 실격 엔딩)의 전략적 타당성을 방증하며, 정석 1스톱을 구사해 우승을 차지했던 2025년 맥라렌(오스카 피아스트리)의 실제 우승 피트 윈도우와 일치함을 검증함.
